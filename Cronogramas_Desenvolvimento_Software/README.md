Cronogramas em Desenvolvimento de Software

Este repositório contém um material de estudo sobre cronogramas em projetos de desenvolvimento de software.

O conteúdo aborda a importância do planejamento de prazos, atividades, dependências, marcos e controle do andamento de um projeto. Também apresenta técnicas de estimativa, como analogia, julgamento especializado, bottom-up, pontos de função, story points e COCOMO.

Principais temas estudados:

- Conceito de cronograma
- Finalidade do cronograma em projetos de software
- Ciclo de vida do software
- Estrutura Analítica do Projeto - EAP/WBS
- Técnicas de estimativa
- Dependências entre atividades
- Caminho crítico
- Diagrama de Gantt
- Marcos do projeto
- Gestão de recursos
- Riscos em cronogramas
- Métodos tradicionais e ágeis
- Indicadores de controle, como PV, EV, AC e SPI

O material também traz exemplos práticos, como sistema acadêmico, sistema de biblioteca e sistema hospitalar, ajudando a entender como montar e acompanhar um cronograma de forma mais realista.

Objetivo do repositório:

Organizar os estudos da disciplina de Engenharia de Requisitos e Análise de Sistemas, registrando conceitos importantes sobre planejamento e controle de cronogramas em projetos de software.

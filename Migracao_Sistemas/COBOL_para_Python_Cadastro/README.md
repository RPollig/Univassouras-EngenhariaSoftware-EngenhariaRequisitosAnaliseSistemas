# COBOL para Python - Sistema de Cadastro

## Linguagem Original
COBOL

## Linguagem Migrada
Python

## Objetivo do Sistema
O sistema realiza cadastro simples de informações utilizando lógica de sistema legado.

## Funcionalidades
- Receber dados do usuário
- Processar informações
- Exibir dados cadastrados
- Encerrar sistema

## Requisitos Funcionais

RF01 - O sistema deve permitir entrada de dados.

RF02 - O sistema deve processar informações cadastradas.

RF03 - O sistema deve exibir informações processadas.

RF04 - O sistema deve encerrar corretamente.

## Requisitos Não Funcionais

RNF01 - O sistema deve preservar lógica do sistema legado.

RNF02 - O código migrado deve ser simples.

RNF03 - O sistema deve possuir comentários explicativos.

RNF04 - O sistema deve facilitar entendimento da migração.

## Migração

O sistema original foi desenvolvido em COBOL e migrado para Python utilizando adaptação da lógica do sistema legado.

## Engenharia Reversa

Foi realizada análise do código COBOL original para entendimento do funcionamento antes da migração para Python.

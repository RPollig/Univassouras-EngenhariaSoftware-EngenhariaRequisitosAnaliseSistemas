# C para Python - Sistema de Funcionários

## Linguagem Original
C

## Linguagem Migrada
Python

## Objetivo do Sistema
O sistema realiza o cadastro e gerenciamento simples de funcionários.

## Funcionalidades
- Cadastrar funcionário
- Listar funcionários
- Buscar funcionário
- Exibir salário

## Requisitos Funcionais

RF01 - O sistema deve permitir cadastrar funcionários.

RF02 - O sistema deve permitir listar funcionários cadastrados.

RF03 - O sistema deve permitir buscar funcionários.

RF04 - O sistema deve armazenar salários dos funcionários.

## Requisitos Não Funcionais

RNF01 - O sistema deve funcionar em terminal.

RNF02 - O sistema deve possuir mensagens claras.

RNF03 - O código deve ser organizado em funções.

RNF04 - O sistema deve permitir manutenção simples.

## Migração

O sistema original foi desenvolvido em linguagem C e migrado para Python utilizando adaptação da lógica original e simplificação da sintaxe.

## Engenharia Reversa

Foi realizada análise do código original para entendimento da lógica de funcionamento antes da migração.

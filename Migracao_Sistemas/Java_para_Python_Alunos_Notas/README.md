# Java para Python - Sistema de Alunos e Notas

## Linguagem Original
Java

## Linguagem Migrada
Python

## Objetivo do Sistema
O sistema realiza cadastro de alunos e gerenciamento de notas.

## Funcionalidades
- Cadastrar aluno
- Registrar nota
- Buscar aluno
- Exibir notas cadastradas

## Requisitos Funcionais

RF01 - O sistema deve permitir cadastrar alunos.

RF02 - O sistema deve permitir registrar notas.

RF03 - O sistema deve permitir buscar alunos.

RF04 - O sistema deve exibir notas cadastradas.

## Requisitos Não Funcionais

RNF01 - O sistema deve funcionar em terminal.

RNF02 - O sistema deve possuir mensagens claras.

RNF03 - O código deve ser organizado.

RNF04 - O sistema deve facilitar manutenção futura.

## Migração

O sistema original foi desenvolvido em Java e migrado para Python utilizando adaptação da lógica e simplificação da sintaxe.

## Engenharia Reversa

Foi realizada análise da estrutura Java para entendimento da lógica antes da migração para Python.

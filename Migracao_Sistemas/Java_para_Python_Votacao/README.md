# Java para Python - Sistema de Votação

## Linguagem Original
Java

## Linguagem Migrada
Python

## Objetivo do Sistema
O sistema realiza uma votação simples, permitindo registrar votos e exibir resultados.

## Funcionalidades
- Registrar votos
- Exibir opções de votação
- Contabilizar votos
- Mostrar resultado final

## Requisitos Funcionais

RF01 - O sistema deve permitir registrar votos.

RF02 - O sistema deve exibir opções disponíveis.

RF03 - O sistema deve contabilizar votos corretamente.

RF04 - O sistema deve mostrar resultado final da votação.

## Requisitos Não Funcionais

RNF01 - O sistema deve funcionar em terminal.

RNF02 - O sistema deve possuir mensagens claras.

RNF03 - O código deve ser organizado.

RNF04 - O sistema deve possuir lógica simples e de fácil entendimento.

## Migração

O sistema original foi desenvolvido em Java e migrado para Python utilizando adaptação da lógica e simplificação da sintaxe.

## Engenharia Reversa

Foi realizada análise do código Java original para entendimento da lógica antes da migração para Python.

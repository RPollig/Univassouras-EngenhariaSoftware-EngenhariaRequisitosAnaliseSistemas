# Migração de Sistemas para Python

Projeto desenvolvido para a disciplina de Engenharia de Requisitos e Análise de Sistemas.

## Objetivo

Realizar a migração de sistemas desenvolvidos em diferentes linguagens para Python, aplicando conceitos de:

- Engenharia Reversa
- Refatoração
- Documentação
- Requisitos Funcionais e Não Funcionais
- Migração de Sistemas Legados

## Sistemas Trabalhados

- Sistema de Funcionários
- Sistema de Alunos e Notas
- Sistema de Votação
- Sistema legado COBOL migrado para Python

## Tecnologias Utilizadas

- Python
- Java
- C
- COBOL
- OnlineGDB
- GitHub

## Observações

Os sistemas foram organizados separadamente para facilitar o entendimento da lógica, documentação e processo de migração.
